# 🐍 Curso Introductorio a Python Científico

[![Python](https://img.shields.io/badge/Python-3.10%2B-blue?logo=python)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

Bienvenido al curso introductorio de **Python Científico**, diseñado para investigadores, estudiantes y profesionales en ciencias biológicas, química y áreas afines que quieran aprovechar Python como herramienta de análisis y visualización de datos.

---

## 🎯 Objetivos

- Familiarizarse con el ecosistema científico de Python.
- Manipular y analizar datos con **NumPy** y **Pandas**.
- Crear visualizaciones claras con **Matplotlib** y **Seaborn**.
- Realizar análisis estadísticos con **SciPy**.
- Aplicar estas herramientas en contextos de bioinformática y biología computacional.

## ⚙️ Instalación

### Con Conda (recomendado)

```bash
conda env create -f environment.yml
conda activate python-cientifico
jupyter lab
```

### Con pip

```bash
pip install numpy pandas matplotlib seaborn scipy jupyterlab
```
